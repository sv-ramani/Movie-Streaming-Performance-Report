# 🎬 Movie Streaming Performance Report | Power BI Dashboard

## 📖 Overview

The **Movie Streaming Performance Report** is an interactive Power BI dashboard that provides insights into the performance of a movie streaming platform. It analyzes key business metrics such as revenue, profit, quantity, discounts, order status, and regional performance. The dashboard is designed to help users identify trends, evaluate business performance, and support data-driven decision-making through interactive visualizations.

---

## 🎯 Project Objectives

- Analyze overall streaming platform performance.
- Monitor key performance indicators (KPIs).
- Compare revenue and profit across movie categories.
- Analyze discount distribution by category.
- Track order status and operational performance.
- Evaluate regional performance using geographic visualization.
- Enable interactive data exploration with filters.

---

## 📊 Dashboard Features

- **KPI Cards**
  - Total Revenue
  - Total Profit
  - Total Quantity

- **Interactive Filters**
  - Region
  - Quantity

- **Visualizations**
  - Revenue and Profit by Category
  - Revenue by Category
  - Discount Distribution by Category
  - Order Status Analysis
  - Regional Performance Map
